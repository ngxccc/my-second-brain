# Dev Sentence Structures (Các khung câu mẫu)

## 1. Commenting Code / Commit (Imperative)
Structure: `Verb + Object`
- **Update** the user profile.
- **Fix** the loading bug.
- **Refactor** the login logic.

## 2. Reporting Issues (Speaking)
Structure: `The [Noun] + is + [Adjective/State]`
- The **server** is **down**.
- The **request** is **slow**.
- The **record** is **missing**.

Structure: `I cannot + [Verb] + [Noun]`
- I cannot **access** the database.
- I cannot **import** the library.

## 3. Asking for Help (Speaking)
Structure: `Can you + [Verb] + [Noun]?`
- Can you **review** my code?
- Can you **check** this **record**?
