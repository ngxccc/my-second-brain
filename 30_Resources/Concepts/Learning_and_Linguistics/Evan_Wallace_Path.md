# Evan Wallace Career Path

## TL;DR

Evan Wallace là co-founder & cựu CTO Figma, người xây dựng rendering engine, multiplayer collab và plugin system. Ông nổi tiếng với esbuild (bundler JS/CSS cực nhanh) sau khi rời Figma năm 2021. Con đường: Brown University CS → Microsoft/Pixar → Figma (2012) → open source legend. Thành tựu dựa trên chuyên môn sâu WebGL, graphics, compilers.

## Core Concept / Rules / Rationales

- **Nền tảng học thuật**: Brown University, CS major. Gặp Dylan Field khi làm TA. Chia sẻ passion WebGL và browser-based collaborative tools.
- **Kinh nghiệm sớm**: Software engineer tại Microsoft và Pixar → tích lũy graphics, rendering, performance.
- **Figma (2012-2021)**: CTO, xây dựng core architecture:
  - Rendering engine (WebGL).
  - Multiplayer collaboration service.
  - Vector/text editors.
  - Plugin system.
  - Quyết định pragmatic giúp scale mà không technical debt quá mức.
- **Sau Figma**: Tập trung open source, nổi bật **esbuild** (2020) – bundler nhanh gấp nhiều lần, thay đổi build pipeline web dev. Được Forbes 30 Under 30 (2017), Inc. Rising Star (2019).
- **Quy tắc thành công**: Chuyên môn sâu một lĩnh vực (graphics + compilers) + timing đúng (browser graphics boom) + execution mạnh (Figma scale). Không phải "genius overnight" mà tích lũy 10+ năm.

## Practical Implementation

**Timeline thực tế**:

- ~2010s: University + early jobs (Microsoft/Pixar).
- 2012: Figma founding (age ~25-28).
- 2017: 30 Under 30.
- 2020: esbuild release.
- 2021: Step back from CTO.
- 2026: Legend trong graphics, compilers, web performance.

**Bài học cho developer**: Tập trung sâu vào niche (ví dụ graphics/WebGL hoặc performance tooling) thay vì generalist. Xây dựng side project thực tế (như esbuild) có thể dẫn đến recognition lớn hơn job title.
