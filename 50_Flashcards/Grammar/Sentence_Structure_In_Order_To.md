Cấu trúc chỉ mục đích "Để / Nhằm mục đích làm gì đó" (In order to / So as to). Nêu công thức và ví dụ.

---

**Cấu trúc chỉ mục đích:**
`In order to / So as to + V-infinitive`
hoặc đặt trong câu:
`S + V + in order to / so as to + V-infinitive`

**Ý nghĩa & Cách dùng:**

- Dùng để diễn tả mục đích của một hành động (làm việc này để đạt được mục tiêu kia).
- Có thể đặt ở đầu câu để nhấn mạnh mục đích hoặc ở giữa câu như một trạng từ chỉ mục đích.

**Ví dụ thực tế:**

- **In order to meet** the deadline, we need to work overtime today. _(Để kịp thời hạn, chúng tôi cần phải làm thêm giờ hôm nay)._
- We optimized the query **so as to reduce** API response time. _(Chúng tôi tối ưu hóa truy vấn để giảm thời gian phản hồi API)._
