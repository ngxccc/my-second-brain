What is a "Defect" (Bug/Fault) under ISTQB, and when does it cause a system failure?

---

A Defect is an imperfection or flaw in a component or system that can cause it to fail to perform its required function. It remains static in code or documents, and only causes a Failure when the specific flawed code path is executed.
