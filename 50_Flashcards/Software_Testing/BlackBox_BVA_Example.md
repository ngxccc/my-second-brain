Design 2-value BVA test cases for an input field that accepts quantities from 1 to 100 (inclusive integers).

---

The boundary points are 1 and 100.

- For boundary 1: 1 (valid boundary) and 0 (invalid neighbor).
- For boundary 100: 100 (valid boundary) and 101 (invalid neighbor).
- Resulting test cases: 0, 1, 100, 101.
