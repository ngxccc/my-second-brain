Compare 2-value BVA vs 3-value BVA and explain when to use each.

---

2-value BVA tests the boundary value and its closest neighbor in the adjacent partition. 3-value BVA tests the boundary, the value just below, and the value just above. 3-value BVA is used for higher-risk systems where relational operator errors must be detected with high confidence.
