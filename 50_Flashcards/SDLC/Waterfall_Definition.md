Define the Waterfall Model and list its core characteristics.

---

The Waterfall Model is a linear-sequential software development life cycle (SDLC) model where development flows steadily downwards.

Core characteristics:

- Each phase must be 100% completed and approved before the next begins.
- Document-driven approach.
- No overlapping or moving backward to previous phases.
